# 🏗️ SIGMA - ERP Fullstack para Empresas Constructoras

¡Bienvenido a SIGMA! Un sistema de gestión integral diseñado para optimizar los flujos operativos, inventarios de almacén y control horario de personal en el rubro de la construcción.

## 📺 Demostración del Sistema en Vivo

![Dashboard-Sigma]([https://github.com](https://github.com/Blem2011/Sigma/issues/1#issue-4646696840))

## 💻 Stack Tecnológico
* **Frontend**: React, TypeScript, Vite, CSS3 Premium.
* **Backend**: Node.js, Express, TypeScript.
* **Base de Datos**: MySQL (Alojada en la nube de Railway).
